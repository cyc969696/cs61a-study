# CS61A Study

This repository documents my CS61A learning journey using Python.
Lesson origin: https://cs61a.org/

## Structure

- `notes/`: weekly notes, lecture notes, discussion notes, summaries
- `cheatsheets/`: distilled concept reviews
- `lab/`: CS61A labs
- `hw/`: CS61A homework
- `proj/`: CS61A projects
- `exam/`: exam preparation
- `practice/`: extra exercises and debugging practice

## Environment

- Python 3.11
- Conda environment: `CS61A-2026`

## Workflow

1. Learn lecture/discussion material
2. Write weekly notes
3. Finish lab / homework
4. Run local tests with `ok`
5. Commit progress to GitHub
